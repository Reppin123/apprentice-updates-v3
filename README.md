# apprentice-updates-v3

Sparkle auto-update feed for **Apprentice F&F v0.3+** (the v5 perception pipeline track).

- `appcast.xml` — the Sparkle feed. v0.3+ clients poll the raw URL.
- Release `.zip` artifacts are attached to GitHub Releases tagged `v<version>` on this repo.

Kept separate from the 0.2.x `apprentice-updates` feed so 0.2.1 clients never auto-update to 0.3.
